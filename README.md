## 📦 Softwares Principais

* **WM:** Sway
* **Barra:** Waybar
* **Terminal:** foot
* **Lançador:** Rofi Wayland

## 🚀 Instalação Rápida

```bash
git clone https://github.com/efzevios/Sway.git
cd Sway
cp -r .config/* ~/.config/
mkdir -p ~/.local/bin
cp -r .local/bin/* ~/.local/bin/
```

## 🤝 Projetos e Complementos Recomendados

Abaixo estão alguns projetos incríveis que recomendo e que complementam perfeitamente este rice:

### Estética
* **[Tela circle icon](https://github.com/vinceliuice/Tela-circle-icon-theme)**
* **[Orchis theme](https://github.com/vinceliuice/Orchis-theme)**

### TUI
* **[Goanime](https://github.com/alvarorichard/GoAnime)**: buscador de animes.
* **[Tldr](https://tldr.sh/)**: cola de comandos, "cheatsheets", colaborativo.
* **[Noctavox](https://github.com/Jaxx497/NoctaVox)**: reprodutor local de músicas. Recomenda-se utilizar o tema swayfx.
* **[Multranslate](https://github.com/Lifailon/multranslate):** tradutor de texto múltiplo. É necessário uso de APIs. Em caso de não adaptação, recomenda-se o [Gtt](https://github.com/eeeXun/gtt).
* **[Tuidict](https://github.com/404Simon/tuidict)**: dicionário multilíngua baseado em [FreeDict](https://freedict.org/).
* **[Gophertube](https://github.com/krishnassh/gophertube)**: buscador e reprodutor (via MPV) de vídeos do Youtube.
* **[Hygg](https://github.com/kruseio/hygg)**: leitor de documentos em interface TUI.
* **[Helm](https://github.com/0xjuanma/helm)**: relógio pomodoro simples.
* **[Awesome TUIs](https://github.com/rothgar/awesome-tuis/)**: paraíso dos projetos TUI.

### Diversos
* **[Spicetify](https://spicetify.app/)**
* **[Linuxtoys](https://github.com/psygreg/linuxtoys)**: projeto com coleções de ferramentas simplificadas para uso de Linux.
